# Athletics Inventory Management System

# Install

Requires Node: https://nodejs.org/en/download/

**Frontend**

```bash
> cd .\frontend
> npm install
> npm start
```

Visit http://localhost:3000 in browser.

**Backend**

```bash
> cd .\backend
> npm install 
> npm start
```

Visit http://localhost:5000 to get started.

**One and Done**

In head directory, run *startApp.bat*
