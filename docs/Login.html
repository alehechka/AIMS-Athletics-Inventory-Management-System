Class: Login

Login(props)

This Component contains the login page along with user authorization logic. State variables: credentials - json - stores the json response from login request. email - string - email of the user password - string- password entered by the user invalid - bool - state variable keeping track of invalid attempts by user noOfAttempts - int - number of login attempts made by the user
