Class: App

App()

This main component mainly does the authentication and routing. State variables: authorized - bool - user authorization. email - string- email address of the authorized user. username - string - username of the authorized user. role - JSON - role of the authorized user. Props passed down from Snackbar provider. enqueuesnackbar - function - shows a snackbar. closesnackbar - function - closes a snackbar.
