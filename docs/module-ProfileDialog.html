Module: ProfileDialog

(require("ProfileDialog"))(props)

Contains the elements in Dialog box form. Props Passed down from Staff: dialogOpen - contains the state of dialog box dialogTitle - contains the title for dialog box inputs - Object containing all form data to be modified sport - list of sports selected in form sports - object containing all sports available sportIdLookup - Lookup for Sport Objects
