Signup(props)

This Component contains the sign up page along with sign up logic. State variables: credentials - json - stores the json response from sign up request. username - string - username email - string - email of the user password1 - string- password entered by the user password2 - string- password reentered by the user password1Valid - bool - state variable keeping track of password length password2Valid - bool - state variable keeping track of password matching
