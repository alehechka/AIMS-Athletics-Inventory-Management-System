# backend

## Setup

1. Go to project folder

> cd backend

2. Install Dependencies

> npm run setup

3. Create .env with following content

````
DB_HOST=
DB_DATABASE=
DB_USER=
DB_PASSWORD=

PORT=3000
````

## Run

1. Start Server

> npx nodemon (resets server when file changes are saved)

2. Open in url or using an Endpoint tester (postman)

> http://localhost:3000/
